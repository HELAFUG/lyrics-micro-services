# Music Service

This is a music service built with the following technology stack:

* FastAPI: A modern, fast (high-performance), web framework for building APIs with Python 3.7+ based on standard Python type hints.
* Redis: An in-memory data store that can be used as a database, cache, message broker, and more.
* Postgres: A powerful, open-source object-relational database system.
* RabbitMQ: A message broker that provides a reliable, scalable, and secure way to send and receive messages.
* React: A JavaScript library for building user interfaces.
